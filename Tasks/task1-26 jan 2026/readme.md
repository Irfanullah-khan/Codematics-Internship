# Task 1 — Semantic HTML Layout (26 January 2026)

## Project Overview

This task was assigned as part of the Codematics Internship Program.
The requirement was to build a complete webpage layout using only semantic HTML5 tags and optimized CSS (no `<div>` tags allowed).

---

## Project Description

The designed webpage layout includes the following sections:

* Header
* Navigation Bar
* Left Sidebar
* Main Content Area
* Right Sidebar
* Footer

### Semantic HTML elements used:

```
header, nav, main, section, aside, footer
```

This ensures a clean, accessible, and well-structured layout.

---

## Technologies Used

* **HTML5 Semantic Tags** – Structure of the layout
* **CSS3 (Flexbox)** – Layout & styling
* **Git & GitHub** – Version control and submission

## Features Implemented

* Fully semantic HTML structure
* Clean and optimized Flexbox-based layout
