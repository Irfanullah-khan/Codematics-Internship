#  **Catchub Landing Page (Task 2)**

##  **Project Overview**

The goal of Task 2 was to recreate a given UI layout **exactly as shown in the reference image**, using only:

* **HTML**
* **CSS**
* **Images provided** in the assets folder

This project demonstrates skills in **UI reproduction, pixel-perfect layout creation, CSS structuring, and clean code formatting**.

###  **Key Features**

* Pixel-perfect replica of the provided Catchub landing page design.
* Fully structured UI sections: Hero, Channels, Messages, Future, and Footer.

* High-quality app mockup images integrated exactly as shown in the reference.
* Professional typography, spacing, and visual hierarchy.
* Ready for extension into a fully responsive landing page.
