##  **Features Implemented**

* Red diagonal hero section built using CSS clip-path.
* Clean layout with titles, logo, and descriptive text.
* Award sections created using reusable HTML components.
* Flexbox used to align icons, headings, and club logos perfectly.
* Club items displayed in an evenly spaced horizontal row.
* Final announcement section styled with matching diagonal design.
