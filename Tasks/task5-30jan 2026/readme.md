##  Project Overview

**Task 5** is a fully responsive web page developed using **HTML, CSS, and Tailwind CSS**. The project replicates a professional law firm website layout and focuses on accurate UI implementation, responsive behavior, and clean structure.

The page was first built using **custom CSS**, and later fully converted to **Tailwind CSS** while maintaining the exact layout and responsiveness.

The goal of this task was to create a **pixel-accurate, responsive layout** matching the provided design reference.

---

##  Features

* Fully responsive layout for mobile, tablet, and desktop
* Pixel-matching layout implementation
* Mobile navigation menu toggle
* Structured multi-section landing page
* Reusable UI components
* Tailwind CSS utility-based styling
* Clean layout alignment and spacing
* Responsive banners, forms, cards, and sidebar sections
