#  Codemiatics Internship — Task 6

## Responsive Portfolio Website (HTML + Tailwind CSS)

This project was developed as part of **Codemiatics Internship – Task 6**.
It is a fully responsive personal portfolio website built using **HTML**, **Tailwind CSS**, and **JavaScript**.

The goal of this task was to create a professional portfolio layout with clean UI, responsive design, and modern web practices.

---

##  Project Overview

A complete developer portfolio including:

* Hero section
* About section
* Skills grid
* Experience timeline
* Projects / Work showcase
* Testimonials
* Contact section
* Footer

The design follows a **mobile-first responsive approach** and adapts to:

* Mobile devices
* Tablets
* Laptops
* Large screens

---

##  Technologies Used

* HTML5
* Tailwind CSS (CDN)
* JavaScript (for dark/light theme toggle)

---

##  Dark Mode Feature

* Implemented using Tailwind’s `dark` class
* Toggle button switches between:

  * Light Mode
  * Dark Mode
* UI updates dynamically across all sections

---

## Responsive Design

Tailwind breakpoints used:

* `sm` → Mobile landscape
* `md` → Tablet
* `lg` → Laptop
* `xl` → Desktop

Layouts adapt automatically:

* Navigation adjusts
* Hero stacks on mobile
* Skills grid reflows
* Experience cards resize
* Projects become single-column
* Testimonials stack
* Contact section centers

---

##  Key Features

* Clean and modern UI
* Utility-first Tailwind styling
* Consistent spacing & typography
* Component-like section structure
* Mobile-first design
* Minimal custom CSS
* Optimized layout hierarchy
