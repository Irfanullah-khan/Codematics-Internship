# JavaScript DOM Tasks

A collection of beginner-to-intermediate **JavaScript DOM-based Tasks** built using **HTML, CSS, and JavaScript**.
This project demonstrates practical use of DOM manipulation, event handling, user input processing, and real-time UI updates.

Designed for learning, practice, assignments, and portfolio showcasing.

---

##  Project Overview

This project contains **7 interactive JavaScript tasks** combined into a single structured application using:

* `index.html`
* `style.css`
* `script.js`

Each task is implemented using clean UI, reusable CSS classes, and organized JavaScript logic.
