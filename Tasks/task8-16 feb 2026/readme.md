# TASK 8— Frontend JavaScript task

##  Introduction

This repository/document presents a collection of frontend development projects completed as part of the **Codematics Internship Program**. The primary objective of these tasks was to strengthen practical skills in **JavaScript, Tailwind CSS, and modern UI development** by building real-world, interactive web applications.

Each task focuses on implementing core programming concepts such as DOM manipulation, event handling, state management, conditional logic, and data persistence using Local Storage.

---

##  Internship Objectives

* Apply JavaScript concepts in real-world applications
* Develop responsive user interfaces using Tailwind CSS
* Build interactive and user-friendly web components
* Understand application logic and UI behavior
* Practice problem-solving through project-based learning

---

#  Task 1 — Calculator Application

### Overview

A responsive calculator capable of performing basic arithmetic operations using JavaScript and Tailwind CSS.

### Key Features

* Addition, subtraction, multiplication, and division
* Clear and delete functions
* Interactive button-based UI
* Responsive Tailwind layout

### Skills Applied

* DOM manipulation
* Event handling
* Functional programming logic
* UI design principles
